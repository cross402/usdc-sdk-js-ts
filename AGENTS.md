# AGENTS.md

Guidance for AI coding agents working on the Agent Tech Pay SDK.

## Project Overview

- **Package**: `@agenttech/pay` — TypeScript/JavaScript client for the Agent Tech v2 payment API
- **Entries**:
  - `src/index.ts` — default entry; exports both `PayClient` and `PublicPayClient`
  - `src/server.ts` — server-side only; `PayClient` (uses `apiKey` + `secretKey`)
  - `src/browser.ts` — client-side only; `PublicPayClient` (no secret credentials)
- **CLI**: `src/cli/index.ts` — binary `agent-pay`; auth config + intent operations
- **Output**: Dual ESM + CJS in `dist/esm` and `dist/cjs`; CLI in `dist/cli`; TypeScript declarations generated by tsc

## Dev Environment

- **Runtime**: Node.js ≥18 or Bun
- **Package manager**: npm, pnpm, or bun
- **TypeScript**: Strict mode; `moduleResolution: "bundler"` for dev, `"node"` for CJS build

## CLI Config

- **Path**: `~/.agent-tech-pay/config.json` (sessions in `~/.agent-tech-pay/sessions/`)
- **Fields**: `apiKey`, `secretKey`, `baseUrl`
- **Commands**: `auth set` writes config; `auth show` displays (secret masked); `auth clear` removes file

## Build

```bash
bun run build
```

- Uses **Bun bundler** for ESM, CJS, and CLI JavaScript output (`--packages external` keeps deps external)
- Uses **tsc** for `.d.ts` declaration files only
- Writes `dist/cjs/package.json` with `{"type":"commonjs"}` for Node CJS resolution

## Test

```bash
bun run test
# or
npm run test
```

- **Vitest** for unit tests in `tests/`
- Run typecheck: `bun run typecheck` or `tsc --noEmit`

## Code Conventions

- **Imports**: Use `.js` extension in source imports (e.g. `from "./client.js"`) for ESM compatibility
- **API shape**: Snake_case in JSON request/response; camelCase in TS via `camelcase-keys` / `decamelize-keys`
- **Errors**: `PayApiError` for HTTP errors, `PayValidationError` for pre-request validation
- **Clients**: `PayClient` (authenticated, `/v2`) and `PublicPayClient` (unauthenticated, `/api`)

## Git & Commits

- **Commit prefix**: Angular-style — `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`
- **Title**: ≤50 characters
- **Message**: English

## File Layout

```
src/              # Source
  index.ts        # Default entry (both clients)
  server.ts       # Server entry (PayClient only)
  browser.ts      # Client entry (PublicPayClient only)
  client.ts       # PayClient & PublicPayClient
  auth.ts         # buildAuthHeaders
  http.ts         # HTTP layer
  errors.ts       # PayApiError, PayValidationError
  types.ts        # Intent types
  utils.ts        # key conversion
  cli/            # CLI
    index.ts      # Entry (shebang + commander)
    config.ts     # ~/.agent-tech-pay/config.json read/write
    commands/
      auth.ts     # auth set / show / clear
      intent.ts   # intent create / execute / get / submit-proof
tests/            # Vitest tests
examples/         # Example scripts (e.g. basic.ts)
dist/             # Build output (gitignored)
  esm/            # ESM output
  cjs/            # CJS output
  cli/            # CLI binary
```
